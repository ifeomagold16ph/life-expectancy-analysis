## Life Expectancy Analysis (2000–2015)

This project performs an exploratory data analysis (EDA) on World Health Organization (WHO) global life expectancy data from 2000 to 2015.

### Objectives
- Analyze trends in life expectancy over time  
- Examine health inequalities between developed and developing countries  
- Investigate the impact of GDP and healthcare expenditure on life expectancy  

### Tools and Libraries
- Python  
- Pandas  
- NumPy  
- Matplotlib  

### Key Steps
- Data cleaning and preprocessing  
- Exploratory data analysis (EDA)  
- Visualization of trends and relationships  

### Summary
This analysis provides insights into global health patterns and highlights socioeconomic factors influencing life expectancy.

### Author
This project was developed as part of a data analysis portfolio in public health.
